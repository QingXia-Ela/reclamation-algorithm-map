将节点连接在一起

在右键菜单点击 进入连线模式 来开始连线，进入后右上角会进行操作提示。

点击第一个节点作为起点，并点击下一个节点来在两个节点之间进行连线

红色节点代表该节点在新创建的线条路径上；蓝色节点代表当前所在的节点，下一个点击的节点会与该节点连接。

在连线模式下按下右键进行连线操作撤销。

最后按下 Esc 退出连线模式即可。

注意！！！

目前还未实现连线删除功能，连线时需谨慎！！！或者参考 地图数据解释 进行删除操作。
