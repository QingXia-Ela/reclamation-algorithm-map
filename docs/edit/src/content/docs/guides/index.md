**我们强烈推荐使用电脑进行地图制作！！！所有的组件与都是根据PC实现的！**

在开始制作前，请检查你的电脑是否能带的动网站：

- 试试用鼠标移动地图，看看是否足够流畅
- 如果发现很卡，开启任务管理器，查看你的 CPU 是不是跑满了，如果是的话可以考虑换台有显卡的电脑（逃）

如果你的体验没问题，这证明你可以开始正常制作地图了！
